We want a language that’s open source, with a liberal license.

Queremos un lenguaje que sea de codigo abierto, con una licencia liberal.

We want the speed of C with the dynamism of Ruby.

Queremos la velocidad de C con el dinamismo de Ruby.

We want a language that’s homoiconic, with true macros like Lisp, but with obvious, familiar mathematical notation like Matlab.

Queremos un lenguaje que sea homoiconico,

Queremos un lenguaje que sea homoiconico, con macros verdaderos como Lisp, pero con la familiar notacion matematatica como Matlab

We want something as usable for general programming as Python, as easy for statistics as R, as natural for string processing as Perl, as powerful for linear algebra as Matlab, as good at gluing programs together as the shell.

Queremos que sea utilizable para programacion general como Python, tan facil para la estadistica como R, tan natural para procesamiento de texto como Perl, tan poderoso para algebra lineal como Matlab, tan bueno en unir programas como la shell.

Something that is dirt simple to learn, yet keeps the most serious hackers happy.

Algo que sea muy simple para aprender, pero que deje feliz a los hackers mas solemnes

We want it interactive and we want it compiled.

Lo queremos interactivo y lo queremos compilado.
